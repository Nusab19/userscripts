### [customSpeedController.js](youtube/customSpeedController.js)

Change video's speed using keyboard.
- `[` : Decreases by 0.1x
- `]` : Increases by 0.1x
- `-` : Decreases by 0.5x
- `+` : Increases by 0.5x
